Supported Versions
==================

Please see https://docs.rapids.ai/install/ for RAPIDS-wide version support.

We aim to meet the `SPEC 0 guidelines <https://scientific-python.org/specs/spec-0000/>`_ for minimal supported versions.

RAPIDS Dependencies
-------------------
nvForest is part of RAPIDS, so its development moves in tandem with the rest of RAPIDS. To ensure compatibility, use the same version of nvForest and other RAPIDS packages. For example, nvForest 26.04 is compatible only with RMM 26.04 and other RAPIDS libraries at version 26.04.
